# Paddock & Paddle

A full-stack business web application built for a local indoor pickleball and equestrian facility in [City, State].

## 🏓 Features

- **Court Booking System** — Browse availability, reserve indoor pickleball courts, and manage bookings
- **Membership Management** — View and purchase membership tiers with integrated payment flow
- **Horse Boarding Portal** — Inquire and manage equestrian care/boarding services
- **User Authentication** — Account creation, login, profile management, and password reset
- **Payment Integration** — Stripe-powered checkout with success/cancel handling
- **Responsive Design** — Mobile-first UI built with Tailwind CSS

## 🛠 Tech Stack

- **Frontend:** React 18, Vite, Tailwind CSS, shadcn/ui
- **Backend & Auth:** Base44 (BaaS platform)
- **Payments:** Stripe
- **Deployment:** Base44

## 🔧 Local Development

```bash
npm install
npm run dev
```

> **Note:** This app requires Base44 environment credentials to run locally. The live version is deployed via Base44.

## 📋 Context

Built as a freelance project for a local small business. Designed, developed, and deployed as a complete business web solution.
